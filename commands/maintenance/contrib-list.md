# wp maintenance contrib-list

Lists all contributors to this release.

Run within the main WP-CLI project repository.

### OPTIONS

[&lt;repo&gt;]
: Name of the repository to fetch the release notes for. If no user/org was provided, 'wp-cli' org is assumed. If no repo is passed, release notes for the entire org state since the last bundle release are fetched.

[&lt;milestone&gt;...]
: Name of one or more milestones to fetch the release notes for. If none are passed, the current open one is assumed.

[\--format=&lt;format&gt;]
: Render output in a specific format.
\---
default: markdown
options:
  - markdown
  - html
\---
