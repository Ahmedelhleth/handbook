# wp user set-role

Set the user role.

### OPTIONS

&lt;user&gt;
: User ID, user email, or user login.

[&lt;role&gt;]
: Make the user have the specified role. If not passed, the default role is
used.

### EXAMPLES

    $ wp user set-role 12 author
    Success: Added johndoe (12) to http://example.com as author.
