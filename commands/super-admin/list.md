# wp super-admin list

List users with super admin capabilities.

### OPTIONS

[\--format=&lt;format&gt;]
: Render output in a particular format.
\---
default: list
options:

  - table
  - csv
  - json
  - count
  - yaml
\---

### EXAMPLES

    # List user with super-admin capabilities
    $ wp super-admin list
