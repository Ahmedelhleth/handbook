# wp network meta list

<small>Quick links: <a href="https://github.com/issues?q=is%3Aopen+label%3Acommand%3Anetwork-meta-list+sort%3Aupdated-desc+org%3Awp-cli">Github issues</a></small>

List all metadata associated with an object.

&lt;id&gt;
: ID for the object.

[\--keys=&lt;keys&gt;]
: Limit output to metadata of specific keys.

[\--fields=&lt;fields&gt;]
: Limit the output to specific row fields. Defaults to id,meta_key,meta_value.

[\--format=&lt;format&gt;]
: Accepted values: table, csv, json, count. Default: table
